## 二、商业模式学习

### 2.1 今日竞品变现模型深度拆解

#### 模型 A：ShenShu AI — 「Freemium + 会员订阅 AI 问答」

```
免费层                      付费层
─────────────────────      ─────────────────────
BaZi 排盘                  会员订阅（月/年）
10 年大运预测        →     AI 命理顾问无限问答
合婚分析                    
                          特点：Q&A 模式天然产生
                          高频使用 → 高留存 → 低 churn
```

**可借鉴点**：
- 「无限 AI 问答」比「一次性报告」的 LTV 更高
- Q&A 交互产生每日活跃数据，可用于模型优化
- 但需要较强的 AI 能力支撑——不能给出错误答案

#### 模型 B：QiAdvisor — 「玄学超市 + 多品类交叉销售」

```
免费入口                    增值服务
─────────────────────      ─────────────────────
BaZi 计算器          →     付费 BaZi 深度报告
奇门遁甲排盘         →     付费奇门深度解读
每日能量预测         →     个性化风水方案
                          人生教练 1v1
                          
                          特点：品类越多，用户
                          终身价值越高，交叉销售
```

**可借鉴点**：
- 多品类降低单品类获客成本（一个用户可能买 BaZi + 奇门 + 风水）
- 但需要极大的内容和技术投入
- **Destiny Code 当前阶段不应模仿**——先深扎 BaZi

#### 模型 C：IMGGIobal 提出的通用变现框架（2026 行业指南）

今日搜索到的 [IMG Global Infotech 2026 占星 App 变现指南](https://www.imgglobalinfotech.com/blog/how-astrology-apps-make-money) 总结了 4 大收入流：

|--------|------|-------------------|
| **订阅制** | 月/年付，解锁高级功能 | ✅ P0 — $9.99/月会员 |

### 3.3 学到了什么

#### 工程经验

**1. 「一鸡多吃」内容分发体系已上线并稳定运行**

```
1 篇博客
  ├── SEO 文章（Google 索引）
  ├── Facebook 帖（每天 10:00）
  ├── Instagram 帖（每天 14:00）
  ├── Newsletter 内容
  ├── RSS Feed
  └── Pinterest Pin（Buffer 直连，待上线）
```

已验证：FB + IG Graph API 双通道自动发布已跑通。下一步接入 Pinterest Buffer。

**2. API 接入生存法则（三原则）**

| 原则 | 说明 | 案例 |
|------|------|------|
| **免费优先** | 优先用免费 API，付费 API 找替代 | Pinterest → Buffer 替代 |
| **前置验证** | 接入前确认账号类型/权限/费用 | IG 需 Business 账号 |
| **安全第一** | Token 加密存储，不在代码/日志中明文 | Graph API Token 永久有效 |

**3. 部署即监控（血泪教训）**

- Vercel 会清理长期未活跃项目
- metaphysical-landing.vercel.app 已 404 长达 6 天
- Google 对长期 404 的站点会降权
- **必须建立**：每日 `curl -sI` 健康检查 → 异常自动告警
